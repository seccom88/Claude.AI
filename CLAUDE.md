# Claude.AI Configuration Router

This is the main configuration router. Depending on your device, Claude Code will load the appropriate file.

## Device-Specific Configs

- **iPhone / Mobile**: See [CLAUDE-iphone.md](./CLAUDE-iphone.md)
- **Mac**: See [CLAUDE-mac.md](./CLAUDE-mac.md)
- **PC / Windows / Linux**: See [CLAUDE-pc.md](./CLAUDE-pc.md)

## Shared Settings

All devices share a common base. See [CLAUDE-shared.md](./CLAUDE-shared.md).

## How to Use

1. Clone or open this repo (`seccom88/Claude.AI`) on any device.
2. Claude Code will auto-load this `CLAUDE.md`.
3. Refer to the device-specific file for setup instructions and local overrides.
4. Keep `CLAUDE-shared.md` updated with settings that apply everywhere.
