# Claude.AI Configuration Router

This is the main configuration router for mobile devices.

## Device-Specific Configs

- **iPhone / Mobile**: See [CLAUDE-iphone.md](./CLAUDE-iphone.md)

## Shared Settings

All devices share a common base. See [CLAUDE-shared.md](./CLAUDE-shared.md).

## Repositories

- Main config repo: `seccom88/Claude.AI`
- Access on any device via: https://claude.ai/code?repositories=seccom88/Claude.AI

## How to Use

1. Open this repo (`seccom88/Claude.AI`) on your iPhone via the Claude mobile app.
2. Claude will load this `CLAUDE.md` along with the shared and device-specific settings.
3. Keep `CLAUDE-shared.md` updated with settings that apply everywhere.
