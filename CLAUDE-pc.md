# PC / Windows / Linux Config

See [CLAUDE-shared.md](./CLAUDE-shared.md) for shared settings.

## PC-Specific Notes

- Use Claude Code CLI, VS Code extension, or JetBrains plugin.
- Windows: prefer PowerShell or WSL2 (Ubuntu) for terminal.
- Linux: use native bash/zsh terminal.

## Recommended Setup (Windows)

- Install WSL2 for a Linux-like environment.
- Install Claude Code CLI inside WSL2: `npm install -g @anthropic-ai/claude-code`
- Or use the VS Code extension from the marketplace.

## Recommended Setup (Linux)

- Install Claude Code CLI: `npm install -g @anthropic-ai/claude-code`
- Run `claude` in any project directory.
