# Mac Config

See [CLAUDE-shared.md](./CLAUDE-shared.md) for shared settings.

## Mac-Specific Notes

- Use Claude Code desktop app or VS Code extension for best experience.
- Terminal shell: zsh (default on macOS).
- Prefer iTerm2 or the built-in terminal.

## Recommended Setup

- Install Claude Code CLI: `npm install -g @anthropic-ai/claude-code`
- Run `claude` in any project directory to start a session.
- Use `claude --repo seccom88/Claude.AI` to open this config repo directly.
